# BONELAB - Rexmeck Weapon Replacer

# Introduction
The rexmeck item replacer is a Code mod designed to replace all Vanilla Weapons in the Game with their Rexmeck Weapon Counterparts.

# Features
- Replaces Vanilla Weapons with their Rexmeck Weapon counterparts (Also replaces NPCs with Gibbable Dudes)
- features a Menu to disable this (for Fusion or other purposes)

# How to Install
- Drag and Drop Folder into your MelonLoader Mods folder
